Notre algorithme

En bref : Pour vous faire ces suggestions d'offres, notre algorithme se base sur les embauches passées de demandeurs d'emploi qui vous ressemblent.
Notre algorithme utilise les éléments présents dans votre profil (dont l'emploi recherché, la formation, l'expérience et les compétences) pour identifier les demandeurs d'emploi qui vous ressemblent parmi ceux ayant été embauchés dernièrement. Ensuite, l'algorithme analyse les caractéristiques des offres sur lesquelles ces demandeurs d'emploi ont été embauchés dans le passé. Enfin, l'algorithme vous recommande les offres en ligne actuellement qui ressemblent le plus à ces anciennes offres.

Malgré nos efforts pour rendre ce système fiable et précis, il est important de comprendre que les prédictions qu'il génère peuvent comporter une marge d’erreur.
